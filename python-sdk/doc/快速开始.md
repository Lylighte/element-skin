# 快速开始

## 1. 创建 OAuth 客户端

```python
from element_skin_sdk import OAuthClient

oauth = OAuthClient(
    base_url="https://skin.example.com",
    client_id="your-client-id",
    redirect_uri="https://your-app.example.com/callback",
)
```

只有能够安全保存密钥的应用才应该配置 `client_secret`。

```python
oauth = OAuthClient(
    base_url="https://skin.example.com",
    client_id="your-client-id",
    client_secret="your-client-secret",
    redirect_uri="https://your-app.example.com/callback",
)
```

桌面应用、CLI、移动端这类公开客户端通常不应该内置 `client_secret`，应依赖 PKCE。

## 2. 请求用户授权

```python
from element_skin_sdk.permissions import ProfileScopes, TextureScopes

session = oauth.authorization_url(
    [
        ProfileScopes.READ_OWNED,
        TextureScopes.READ_OWNED,
    ],
    state="csrf-state-value",
)

print(session.authorization_url)
```

将用户跳转到 `session.authorization_url`。应用需要保存
`session.code_verifier` 和 `session.state`，等待回调返回。

## 3. 交换授权码

```python
tokens = oauth.exchange_code(
    code="code-from-callback",
    code_verifier=session.code_verifier,
)
```

`tokens` 是一个 `TokenSet`：

```python
tokens.access_token
tokens.refresh_token
tokens.expires_in
tokens.permissions
```

## 4. 调用 API

```python
from element_skin_sdk import ElementSkinAPI

api = ElementSkinAPI("https://skin.example.com", token=tokens)
profiles = api.list_profiles(page_size=20)
```

当 token 中包含权限列表时，SDK 会先做本地权限校验。缺少所需权限时，会在发起
HTTP 请求之前抛出 `PermissionDenied`。

## 5. 刷新 token

```python
new_tokens = oauth.refresh(tokens.refresh_token)
```

如果服务端轮换了 refresh token，应用必须保存新的 `refresh_token`。

## 6. 撤销 token

```python
oauth.revoke(tokens.access_token)
```
